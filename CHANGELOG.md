## ملاحظة مهمة بخصوص الـ AI API

الكود يستدعي Claude API مباشرة. في بيئة المتصفح:
- قد يعمل إذا كان الموقع يمر عبر proxy يضيف الـ API key
- إذا لم يعمل (CORS)، الشات والتحليل يرجعان تلقائياً للمحرك المحلي (fallback)
- للهاكاثون: المحرك المحلي وحده يعطي تجربة كاملة، والـ AI يضيف قيمة إضافية

### لتفعيل AI بشكل كامل:
الخيار الأسهل: أنشئ endpoint بسيط على Vercel Serverless Function يستدعي Claude API ويرجع النتيجة.
